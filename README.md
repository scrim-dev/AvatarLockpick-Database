# AvatarLockpick-Database
Database of known avatar locks/locking systems. 

This database is used in the AvatarLockpick app.

## How do I view the data?
You can view it by using [sqlitebrowser](https://github.com/sqlitebrowser/sqlitebrowser) and downloading this file: [VRC_LOCKS.db](https://github.com/scrim-dev/AvatarLockpick-Database/blob/main/db/VRC_LOCKS.db)
